Learning-Aided INS Drift Suppression for GPS-Denied Quadrotor Flight

Technical Deep Dive

1. Problem Definition

In GPS-denied environments, quadrotors rely solely on inertial measurements for state estimation and control.

IMU bias and noise accumulate over time, causing:

  • Position drift
  • Orientation error
  • Navigation instability
  • Amplified disturbance response

Classical PID and EKF methods assume bounded noise and predictable disturbances. Real-world turbulence violates these assumptions.

2. Research Gap

Existing Systems

  • Fixed-gain PID
  • EKF bias estimation
  • Disturbance observers

Limitations

  • Limited adaptability
  • Static noise models
  • Weak generalization
  • No learning-based bias prediction

3. Phase 3: Disturbance-Resilient Stabilization

Method

  • PPO reinforcement learning
  • Wind-injected simulation
  • Multi-regime training
  • 600k timesteps

Outcome

  • 85% stability under strong wind
  • Reduced angular variance
  • Cross-regime robustness

4. Phase 4: Learning-Aided INS Drift Suppression

Log IMU bias accumulation
Quantify baseline INS+EKF drift
Train sequence model (LSTM / Transformer)
Predict bias corrections (Δbg, Δba)
Fuse correction into EKF
Compare ATE and RPE metrics
Target: ≥ 30% drift reduction.
